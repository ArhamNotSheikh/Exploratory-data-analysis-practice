# 🎢 Roller Coaster EDA

A simple Exploratory Data Analysis project done in Python using a roller coaster dataset.

Built this while learning Data Analytics with Python and practicing real-world EDA workflows like:
- data cleaning
- handling null values
- visualization
- feature analysis
- finding trends in data

## 🛠️ Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 What’s Inside
- Data cleaning & preprocessing
- Datatype conversion
- Duplicate/null value handling
- Visualizations & correlations
- Basic insights from the dataset
